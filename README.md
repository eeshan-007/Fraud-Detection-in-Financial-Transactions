# Fraud Detection in Financial Transactions

## Overview
This project uses Machine Learning to identify fraudulent financial transactions using the IEEE-CIS Fraud Detection Dataset.

## Dataset
- IEEE-CIS Fraud Detection Dataset
- 590,540 transactions
- Transaction and identity features

## Model
- CatBoost Classifier
- Missing value handling
- Class imbalance treatment
- Feature importance analysis

## Results
- ROC AUC: 0.953
- PR AUC: 0.739
- Recall: 84.9%
- Precision: 28.4%

## Technologies
- Python
- Pandas
- Scikit-Learn
- CatBoost
- NumPy
